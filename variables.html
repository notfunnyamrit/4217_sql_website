Variables in PL/SQL

Variable Declraration

**Syntax**
variable_name [CONSTANT] datatype [NOT NULL] [:= | DEFAULT initial_value] 
**EXAMPLE**
Radius Number := 5;
Date_of_birth date;

Explanation:A variable is a meaningful name which facilitates a programmer to store data temporarily during the execution of code. It helps you to manipulate data in PL/SQL programs. It is nothing except a name given to a storage area. Each variable in the PL/SQL has a specific data type which defines the size and layout of the variable's memory.

Constant Declaration

**Syntax**
const_name CONSTANT datatype[NOT NULL]:=expression|value;
**EXAMPLE**
pi CONSTANT NUMBER (9,7) := 3.14;
dept CONSTANT VARCHAR(10) := 'COMPUTER';

Explanation:In the first declaration, pi is constant of NUMBER datatype which is assinged a value 3.14. In the second declaration, dept is declared as constant of varchar2 datatype which is assigned a value COMPUTER.

Variable Atrributes

%TYPE Declaration:

SALARY EMP.SAL %TYPE;

Explanation:In PL/SQL, you can declare variables to be of the same data type as a column in a table or cursor using the %TYPE attribute. This ensures data consistency and reduces the risk of type mismatch errors.

%ROWTYPE

EMPLOYEE EMP%ROWTYPE;

Explanation:This declaration declares a record named employee having fields with the same name and data types as that of columns in the EMP table.

INITAILIZING VARIABLES

Adding two numbers

DECLARE  
   a integer := 30;  
   b integer := 40;  
   c integer;  
   f real;  
BEGIN  
   c := a + b;  
   dbms_output.put_line('Value of c: ' || c);  
   f := 100.0/3.0;  
   dbms_output.put_line('Value of f: ' || f);  
END;

Explanation:We declared 4 variables and then created a simple addition function in BEGIN.
