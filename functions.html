SQL FUNCTIONS

MIN() Function

SELECT MIN(Price) AS SmallestPrice
FROM Products;

Explanation:finds the price of the cheapest product

MAX Function

SELECT MAX(Price) AS LargestPrice
FROM Products;

Explanation:finds the price of the most expensive product

COUNT Function

SELECT COUNT(ProductID)
FROM Products;

Explanation:The COUNT() function returns the number of rows that matches a specified criterion.

AVG Function

SELECT AVG(Price)
FROM Products;

Explanation:The AVG() function returns the average value of a numeric column.

SUM Function

SELECT SUM(Quantity)
FROM OrderDetails;

Explanation:The SUM() function returns the total sum of a numeric column.

IFNULL Function

SELECT ProductName, UnitPrice * (UnitsInStock + IFNULL(UnitsOnOrder, 0))
FROM Products;

Explanation:The MySQL IFNULL() function lets you return an alternative value if an expression is NULL. The example above returns 0 if the value is NULL
