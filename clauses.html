CLAUSES

WHERE Clause

SELECT * FROM employees WHERE department_id = 10;

Explanation: This query filters employees based on the department_id equal to 10.

LIMIT Clause

SELECT * FROM Customers
WHERE Country='Germany'
LIMIT 3;

Explanation:selects the first three records from the "Customers" table, where the country is "Germany"

HAVING Clause

SELECT COUNT(CustomerID), Country
FROM Customers
GROUP BY Country
HAVING COUNT(CustomerID) > 5;

Explanation:lists the number of customers in each country. Only include countries with more than 5 customers

Order By Keyword

SELECT * FROM Customers
ORDER BY Country;

Explanation:selects all customers from the "Customers" table, sorted by the "Country" column

JOIN Clause

SELECT Orders.OrderID, Customers.CustomerName, Orders.OrderDate
FROM Orders
INNER JOIN Customers ON Orders.CustomerID=Customers.CustomerID;

Explanation:A JOIN clause is used to combine rows from two or more tables, based on a related column between them.
The "CustomerID" column in the "Orders" table refers to the "CustomerID" in the "Customers" table. The relationship between the two tables above is the "CustomerID" column.
